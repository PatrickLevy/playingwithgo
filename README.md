### To Run
```
1. install Go

2. Setup your go project directory in your profile (here's how I did it):

export GOPATH=/Users/<name>/<folder-with-projects>/go
export PATH=$PATH:$GOPATH/bin

3. Make a directory for your personal Go projects (here's mine):
/Users/<name>/Development/go/src/github.com/<my-github-name>/

4. Change into the above directory

5. Clone repo
git@github.com:PatrickLevy/playingwithgo.git

6. Change into above Directory

7. Run
go run main.go 
```
